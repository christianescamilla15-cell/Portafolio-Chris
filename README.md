# Portfolio - Christian Escamilla

Full Stack Developer Portfolio profesional con enfoque en IA y sistemas multi-agente.

## 🌐 Preview

**https://christian-escamilla.vercel.app**

## 📁 Estructura

```
portafolio/
├── index.html          # Página principal
├── css/
│   ├── style.css      # Estilos principales
│   └── particles-fallback.css
├── js/
│   ├── main.js        # Funcionalidad interactiva
│   └── particles.min.js
└── img/               # Imágenes de proyectos
```

## ✨ Características

- **Hero Section** - Presentación con typewriter effect
- **Sobre Mí** - Stats animados con años de experiencia
- **Skills** - Grid de 4 categorías tecnológicas
- **Proyectos** - Cards con 6 proyectos destacados
- **Contacto** - Formulario funcional
- **Animaciones** - Scroll animations y partículas
- **Responsive** - Adaptado a móvil, tablet y desktop

- HTML5 / CSS3 / JavaScript (Vanilla)
- Google Fonts (Inter + Fira Code)
- Particles.js para efectos de fondo
- Vercel para hosting

## 🚀 Deploy

```bash
# Instalar Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```
